# Online Banking System

## Features
- User Registration
- User Login
- Dashboard
- Deposit & Withdrawal
- Transaction History
- Bar Chart Visualization
- Pie Chart Visualization
- Line Chart Visualization

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

## How to Run
1. Install XAMPP
2. Import database.sql into phpMyAdmin
3. Place project in htdocs
4. Start Apache and MySQL
5. Open localhost/projectname
