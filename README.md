# Online Banking System

A web-based banking system developed using PHP, MySQL, HTML, CSS, and JavaScript.

## Features
- User Registration
- User Login
- Dashboard
- Deposit & Withdrawal
- Transaction History
- Data Visualization

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

## How to Run
1. Install XAMPP.
2. Import database.sql into phpMyAdmin.
3. Place the project folder in htdocs.
4. Start Apache and MySQL.
5. Open the project in your browser.
